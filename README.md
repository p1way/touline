# Touline
An analytic model for the Touline knot, to try to answer the riddle of a friend : *"Can you guess how long a piece of string I needed to make this knot?"*

My code is largely inspired from this paper --> https://www.mathcurve.com/courbes3d.gb/noeuds/Equations_des_noeuds_09_04_17.pdf
